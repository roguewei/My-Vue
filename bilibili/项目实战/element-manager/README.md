### 项目搭建

```
完全引入element-ui

1、npm安装element ui
npm i element-ui -S

2、在main.js中引入
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI)
```
